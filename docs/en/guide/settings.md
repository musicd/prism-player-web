# Settings

Tune Prism English to your habits so review stays sustainable.

## Language & Goals

- **Target language**: drives dictionary, glosses, and UI hints.
- **Subtitle display**: original, translation, or bilingual (per version).

## Study & Review

| Option | Purpose |
|--------|---------|
| Daily new-word cap | Limits new words per session |
| Daily review cap | Keeps load manageable on busy days |
| SRS strength | More conservative (more reviews) vs more aggressive (longer gaps) |

## Playback & Subtitles

- **Subtitle offset**: fix external subs that drift from video.
- **Default track**: preferred language on multi-track files.
- **Playback speed**: slower while learning; normal speed when replaying on review.

## Shortcuts (reference)

Exact bindings are in-app; common actions:

- Play / pause current sentence clip
- Next / previous sentence
- Known / unknown
- Reveal answer; play original sentence during review

## Data & Backup

Progress, vocabulary, and cards live in a local user data folder by default. Back up that folder regularly (path in About); especially before reinstalling the OS.
