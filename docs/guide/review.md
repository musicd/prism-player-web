# 复习与句子回放

复习是词有戏的核心闭环：间隔重复帮你对抗遗忘，**句子回放**帮你保留语境。

## 复习队列从哪来？

所有在学习中标记为「不认识」的单词会进入复习队列。系统根据间隔重复（Spaced Repetition）安排复习时间，例如：

- 第 1 次：学习后约 1 天
- 第 2 次：约 3 天
- 第 3 次：约 7 天
- 之后：约 30 天及更长间隔（具体算法以应用内为准）

每次你正确回忆，间隔会拉长；回忆失败则缩短间隔，增加见面次数。

## 一次复习 session 怎么走？

1. 打开「今日复习」或「待复习」列表；
2. 看到单词（或听发音）——**先尝试回忆**释义与用法；
3. 自行评判是否想起，再点击揭示答案；
4. 点击 **播放原句**：应用跳转到该词首次学习时的视频位置，播放完整句子；
5. 根据掌握程度标记「记得 / 模糊 / 忘了」，系统调整下次复习时间。

## 句子回放为什么重要？

研究表明，词汇在**有意义的语境**中更容易被长期记住。你复习时听到的不仅是发音，还有角色的语气、场景的紧张感、笑点的节奏——这些都是纯词表无法提供的「情感与情境线索」。

若某个词多次复习仍模糊，建议回到 [学习一节内容](/guide/learning-session)，在同一集里多读几句 surrounding 台词，扩大语境暴露。

## 为什么这样设计？

单词离开句子就容易变成抽象符号。词有戏在复习界面保留「回到原句」的一键入口，让每一次认词都重新连接到你看过的那一幕。理念背景：[为何这样学语言](/about/why-learn-this-way)。
