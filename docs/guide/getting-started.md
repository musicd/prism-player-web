# 开始使用

词有戏不是又一个孤立的背单词 App。它从你**已经在看、已经感兴趣**的影视内容出发：字幕是教材，画面是语境，你的每一次「认不认识」都是一次真实的学习。

## 安装（桌面端）

1. 前往 [下载页](/download) 获取 Windows 或 macOS 安装包（链接正式发布后可用）。
2. 按平台提示完成安装并首次启动。
3. 在欢迎页选择你的**目标学习语言**（例如学英语）。

::: warning
请仅使用官方渠道的安装包。视频文件版权由你自行负责，词有戏仅帮助你基于已有字幕进行个人学习。
:::

## 五分钟了解整体流程

```mermaid
flowchart LR
  importMedia[导入视频与字幕]
  learnSession[按句子学习]
  wordCheck[逐词认识与否]
  cards[生成记忆卡片]
  review[间隔复习与句子回放]
  importMedia --> learnSession --> wordCheck --> cards --> review
```

| 步骤 | 说明 |
|------|------|
| 1. 导入 | 添加本地视频，加载内嵌或外挂字幕 |
| 2. 学习 | 按字幕句推进，观看片段并理解句意 |
| 3. 认词 | 将句子拆成单词，标记认识 / 不认识 |
| 4. 卡片 | 不认识的词进入个人词库与复习队列 |
| 5. 复习 | 按间隔重复回顾；播放学过的原句加深记忆 |

详细操作见：

- [导入视频与字幕](/guide/import-media)
- [学习一节内容](/guide/learning-session)
- [复习与句子回放](/guide/review)

## 第一次学习建议

- 选一集你**已经看过、熟悉剧情**的剧集，降低理解句意的认知负担。
- 单次学习 10～20 句即可，质量优于数量。
- 当日产生的「不认识」词汇，隔天用复习功能过一遍。

想了解背后的学习理念，请阅读 [为何这样学语言](/about/why-learn-this-way)。
