# 设置与偏好

根据你的学习习惯调整词有戏，让复习可持续、不挫败。

## 语言与目标

- **目标学习语言**：决定词典、释义与 UI 提示的主语言。
- **字幕显示**：学习时显示原文、译文或双语（视版本支持）。

## 学习与复习

| 选项 | 说明 |
|------|------|
| 每日新词上限 | 限制单次学习会话中新词数量，避免贪多 |
| 每日复习上限 | 控制复习负担，适合忙碌的工作日 |
| 间隔重复强度 | 偏保守（更多复习）或偏激进（更长间隔） |

## 播放与字幕

- **字幕时间轴偏移**：外挂字幕与画面不同步时微调（毫秒级）。
- **默认字幕轨**：多轨视频时指定优先语言。
- **播放速度**：学习时可略慢速，复习原句时可恢复正常语速。

## 快捷键（参考）

具体键位以应用内说明为准，常见包括：

- 播放 / 暂停当前句片段
- 下一句 / 上一句
- 认识 / 不认识
- 复习时揭示答案、播放原句

## 数据与备份

学习进度、词库与卡片数据默认保存在本机用户目录。请定期备份该目录（路径见应用「关于」页），重装系统前尤其重要。
