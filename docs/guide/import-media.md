# 导入视频与字幕

棱镜英语以**本地视频 + 字幕**为学习原料。你可以导入美剧、电影、访谈或任何带字幕的视频文件。

## 支持的视频

将视频文件放入应用内的媒体库，或通过「导入」选择本地路径。常见格式包括 MP4、MKV、MOV 等（以你使用的版本说明为准）。

::: tip
首次使用建议选体积适中、字幕清晰的一集剧集，便于快速体验完整学习流程。
:::

## 字幕来源

| 类型 | 说明 |
|------|------|
| 内嵌字幕 | 视频文件内封装的字幕轨，启动时选择语言轨道即可 |
| 外挂字幕 | 与视频同名的 `.srt`、`.ass` 等文件，放在同一目录或由应用指定路径 |
| 多语言 | 可同时保留中英字幕，学习时切换或对照（视版本功能而定） |

若字幕与时间轴略有偏差，可在设置中微调偏移量（参见 [设置与偏好](/guide/settings)）。

## 拆分为学习材料

导入成功后，应用会：

1. 解析字幕时间轴；
2. 将每一条字幕对应为一个**学习单元**（句子 + 起止时间）；
3. 关联视频片段，供学习时跳转播放。

你无需手动切分——棱镜英语按字幕自动拆分。下一步：[学习一节内容](/guide/learning-session)。

## 合规提醒

- 请使用**合法获得**的视频与字幕文件。
- 棱镜英语不提供、不分发任何影视内容，仅处理你本地的文件。
