# 学习一节内容

一次「学习会话」通常对应一集视频中的一段字幕：按句推进，理解句意，再逐词确认你是否认识。

## 1. 选择学习内容

在媒体库中打开已导入的视频，选择集数或时间范围，进入学习模式。界面会按字幕顺序展示当前句子。

## 2. 理解句子

- 播放与该句对应的视频片段；
- 阅读字幕原文（及可选译文）；
- 尝试**先不看译文**复述大意，再对照确认。

::: info 主动提取
在揭示译文或释义之前，先在心里回答「这句话在说什么？」——这一步能显著提升记忆效果。详见 [为何这样学语言](/about/why-learn-this-way)。
:::

## 3. 逐词：认识还是不认识

句子理解后，应用将句子拆分为单词列表（已过滤常见虚词的情况以版本为准）。对每个词：

| 你的操作 | 结果 |
|----------|------|
| **认识** | 记录为已掌握，不进入复习队列（或降低复习频率） |
| **不认识** | 加入个人词库，并生成记忆卡片 |

请诚实选择。标记「认识」却想不起来的词，复习时反而会浪费次数；标记「不认识」则会在合适的时间再次遇到它。

## 4. 生成记忆卡片

不认识的词会关联：

- 单词拼写与释义（可接入词典，视版本而定）；
- **来源句子**与视频时间戳；
- 首次学习日期，用于间隔复习调度。

完成当前批次后，可在仪表盘查看今日新词数量与待复习数量。

## 为什么这样设计？

孤立背单词很难记住用法和语气。棱镜英语把每个词**钉在它出现的原句里**，你复习时听到的不是例句库里的模板，而是角色真实说过的那句话。下一步：[复习与句子回放](/guide/review)。
