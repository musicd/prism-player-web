features:
  - title: 🎬 视频字幕学习材料
    details: 支持美剧及任意带字幕的视频。按字幕时间轴自动拆成句子单元，每一句都附带画面与音频，学习材料来自你真正想看的内容。
  - title: 🔤 句子到单词
    details: 将字幕句拆成单词，逐词询问「是否认识」。认识的词轻轻划过，不认识的词进入你的个人词库，避免无效背单词。
  - title: 🃏 智能记忆卡片
    details: 不熟悉的单词自动生成间隔复习卡片。系统按记忆曲线安排 1 天、3 天、7 天、30 天后的回顾，让词汇真正留下来。
  - title: 🔊 句子回放复习
    details: 复习单词时，一键播放你当初学习过的原句与片段。单词不再孤立存在，语境与情绪一起被唤醒。
  - title: 🧠 主动提取学习
    details: 先回忆、再揭示——不是被动盯字幕。每一次「认不认识」的判断，都是一次主动提取，比单纯刷剧有效得多。
  - title: 🛡️ 本地优先与隐私
    details: 桌面端架构，视频与字幕在本地处理。你的学习进度与词库保存在本机，无需把私人观影习惯上传到云端。
