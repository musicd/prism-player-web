# 常见问题

## 词有戏是什么？

一款**桌面端**语言学习工具：导入本地视频与字幕，按句学习、逐词标记认识与否，为生词生成间隔复习卡片，复习时可回放学习过的原句。

## 支持哪些视频和字幕？

常见本地视频格式（如 MP4、MKV 等）及内嵌 / 外挂字幕（如 SRT、ASS）。详见 [导入视频与字幕](/guide/import-media)。

## 需要联网吗？

核心学习、复习、播放功能以**本地**为主。若版本包含在线词典、更新检查或反馈上报，会在设置中说明，并可选择关闭（以实际版本为准）。

## 学习数据存在哪里？

默认保存在本机用户数据目录。请自行备份；卸载前导出或复制该目录以免丢失进度。见 [隐私政策](/legal/privacy-policy)。

## 和 Anki 有什么区别？

词有戏围绕**影视字幕流程**设计：按时间轴拆分、学习时跳转视频、复习时回放原句。Anki 是通用记忆工具，二者可互补；数据是否互通取决于未来版本规划。

## 支持学哪些语言？

取决于你导入的字幕与词典配置，目标语言在设置中选择。英语是目前最典型的使用场景，其它语种以词典与分词支持为准。

## 字幕版权有问题吗？

词有戏**不提供**任何影视资源。请仅使用合法获得的视频与字幕文件。应用只处理你本地的文件，不对内容版权承担责任。

[下载页](/download)。Windows 与 macOS 安装包在正式发布后更新。
