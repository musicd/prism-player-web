# 为何这样学语言

很多人学英语的路径是：背单词 App → 放弃 → 刷美剧带字幕 → 觉得「好像在学」却说不清学到了什么。棱镜英语想解决的是中间那片空白：**让你在看懂剧情的同时，真的把台词里的语言带走。**

## 三种常见方式的局限

| 方式 | 优点 | 局限 |
|------|------|------|
| 背单词 App | 系统、可量化 | 词与真实用法脱节，难坚持 |
| 纯刷剧 | 有趣、输入量大 | 被动接收，熟词重复、生词滑过 |
| 课堂教材 | 结构清晰 | 内容常与你个人兴趣无关 |

棱镜英语组合了三者的长处：兴趣来自影视，结构来自字幕拆分与复习计划，深度来自**主动判断每个词认不认识**。

## 我们的方法：字幕 + 主动认词 + 句子回放

### 1. 字幕是天然的「可理解输入」

语言习得研究（如 Krashen 的可理解输入假说）强调：学习者需要略高于当前水平、但**大致能理解**的语言材料。你已经喜欢的美剧，配上字幕与时间轴，正好构成这种材料——比随机词表更接近真实沟通。

### 2. 主动提取优于被动观看

「这个词认不认识？」迫使你**从记忆里检索**，而不是扫一眼字幕就滑过去。检索本身会强化神经通路；即使猜错，纠正后的记忆也更牢。这比「看一百集就算学了」有效得多。

### 3. 间隔重复对抗遗忘曲线

艾宾浩斯遗忘曲线说明：不复习的信息很快淡忘。把不认识的词做成卡片，在 1、3、7、30 天等节点复习，是用最小时间成本维持长期记忆的经典做法。棱镜英语在应用层实现这一逻辑，你只需每天打开「今日复习」。

### 4. 复习时回放原句 = 语境记忆

同一个词在不同句子里意思可能微妙不同。*run* 可以是跑，也可以是经营、流传。你第一次遇到它时的那句台词，就是只属于你个人的最佳例句。复习时一键回放，是在**重建当时的情境**，而不只是背释义。

::: warning 说明
上文涉及的语言学习与记忆规律来自公开学术与科普共识，**不构成医疗或教育诊断建议**。个人效果因基础、投入时间与材料难度而异。
:::

Anki 等工具擅长通用间隔重复；棱镜英语专注**影视字幕工作流**：导入、按句学习、自动关联时间戳、复习时回播片段。若你已是 Anki 重度用户，可将棱镜英语视为「带画面和剧情的专用前端」；是否导出互通以未来版本为准。
